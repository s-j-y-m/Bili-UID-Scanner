# 哔哩哔哩UID遍历扫描工具-Bili-UID-Scanner
这个工具可以帮助你根据UID扫描纯英文/纯英文+数字ID的用户
# 使用方法
运行.exe文件，输入你想要扫的ID段

例如
```
114514-114999
```
这样，工具会帮你扫描114514-114999所有的用户，并且输出纯英文ID/英文+数字混合的用户到控制台和results.txt

**你还可以使用中文引号包住一个变量区域！**
支持格式：
```
- "10"114514-"99"114514 或 '10'114514-'99'114514 (变量在开头)
- 114"514"114-114"999"114 或 114'514'114-114'999'114 (变量在中间)
- 114514"10"-114514"99" 或 114514'10'-114514'99' (变量在结尾)
```
