# 哔哩哔哩UID遍历扫描工具 - Bili-UID-Scanner

这个工具可以帮助你根据 UID 段扫描出**纯英文 ID** 或 **英文+数字混合 ID** 的用户。

## ✨ 功能特点

- 支持指定 UID 范围批量扫描
- 自动筛选出**纯英文** / **英文+数字混合** 的用户名
- 结果同时输出到**控制台**和 `results.txt` 文件
- 支持**变量搜索**，灵活搜索复杂的 UID 模式

## 🚀 使用方法

直接运行 `.exe` 文件，输入你想要扫描的 UID 段。

### 基础用法

例如输入：
```
114514-114999
```

工具会扫描 UID 从 `114514` 到 `114999` 的所有用户，并将符合条件的 ID 输出到控制台并保存到 `results.txt`。

### 高级用法：变量占位符

你可以使用**中文引号** `""` 包裹一个变量区域，灵活搜索 UID。

**支持三种变量位置：**

| 类型 | 格式示例 | 说明 |
| :--- | :--- | :--- |
| 变量在开头 | `"10"114514-"99"114514` | 前两位可变 |
| 变量在中间 | `114"514"114-114"999"114` | 中间三位可变 |
| 变量在结尾 | `114514"10"-114514"99"` | 最后两位可变 |

> **注意**：请使用**中文双引号** `" "`包裹变量部分。

## 📝 输出示例

扫描过程中，符合条件的 UID 和用户名会实时显示在控制台，同时追加到 `results.txt` 文件中。
```
2026-04-21-12-14-31（开始时间）
1-123456-XXXX
2-123789-XXXXXXX
3-114514-XXXXXX
```
